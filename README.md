# Scripts-Bash
## 📌 À propos
Ce dépôt contient des scripts **Bash** développés pour :
- Automatiser des tâches répétitives
- Monitorer des serveurs
- Sécuriser des infrastructures

### Prérequis
Pour exécuter ces scripts, assurez-vous que votre environnement répond aux exigences suivantes :
   **Élément**       | **Exigence**                          | **Vérification**               |
 |-------------------|---------------------------------------|--------------------------------|
 | **Système**       | Linux (Debian, RHEL, etc.)   | `uname -a`                     |
 | **Bash**          | Version 4.0 ou supérieure            | `bash --version`               |
 | **Droits**        | Accès `sudo` (si nécessaire)          | `sudo -v`                      |
